Titanic: Machine Learning from Disaster¶

  • Objective - Create a model that predicts which passengers survived from the Titanic shipwreck
  • Dataset Source - https://www.kaggle.com/c/titanic/data
  • Dataset Format - .csv

Some useful information¶

The sinking of the Titanic is one of the most infamous shipwrecks in history.

On April 15, 1912, during her maiden voyage, the widely considered “unsinkable” RMS Titanic sank after colliding with an iceberg. Unfortunately, there weren’t enough lifeboats for everyone onboard, resulting in the death of 1502 out of 2224 passengers and crew.

While there was some element of luck involved in surviving, it seems some groups of people were more likely to survive than others.

Data Description¶

VariableDefinitionKey
survival Survival 0 = No, 1 = Yes
pclass Ticket class 1 = 1st, 2 = 2nd, 3 = 3rd
sex Sex
Age Age in years
sibsp # of siblings / spouses aboard the Titanic
parch # of parents / children aboard the Titanic
ticket Ticket number
fare Passenger fare
cabin Cabin number
embarked Port of Embarkation C = Cherbourg, Q = Queenstown, S = Southampton

def one_hot_encoding(data, columns):
    nominal_df = []

    for nc in columns:
        onehot_df = pd.get_dummies(data[nc], prefix=nc)
        onehot_df.drop(onehot_df.columns[-1], axis=1, inplace=True)
        
        data.drop(columns=[nc], inplace=True)
        nominal_df.append(onehot_df)

    onehot_combined = pd.concat(nominal_df, axis=1)

    data = data.join(onehot_combined)
    return data

def balance_class(data):
    df = data.copy()
    survived_idx = df[df.Survived==1].index

    df = pd.concat([df.loc[survived_idx], \
                    df.loc[~df.index.isin(survived_idx)].sample(len(survived_idx))])

    df = df.sample(frac=1).reset_index(drop=True)
    return df

def select_best_model(X, y, test_size=0.2, standarize=False, balance=False):
    """
    Find classification model based on some evaluation metrics
    
    Cache contain used training and evaluation data 
    
    model_report describe how is performance for each model
    
    Return - cache, best_model, model_report
    """
    if balance==True:
        df = pd.concat([X,y], axis=1)
        df = balance_class(df)
        y = df['Survived']
        X = df.drop(columns=['Survived'], axis=1)
        
    X_train, X_val, y_train, y_val = train_test_split(X, y, test_size=test_size, random_state=7)
    
    best_model = []
    best_score = 0
    models = [LogisticRegression(max_iter=1000),
              DecisionTreeClassifier(max_depth=25, max_leaf_nodes=5),
              RandomForestClassifier(n_estimators=100, min_samples_leaf=5),
              GradientBoostingClassifier(n_estimators=100),
              XGBClassifier(n_estimators=90),
              AdaBoostClassifier(),
              BaggingClassifier(),
              ExtraTreesClassifier(),
              MLPClassifier(max_iter=1000),
              KNeighborsClassifier(n_neighbors=5)]
    
    f1_scores = []
    roc_auc_scores = []
    train_scores = []
    accuracy_scores = []
    model_names = []

    for model in models:    
        model.fit(X_train, y_train)
        
        y_predicted = model.predict(X_val)
        score = f1_score(y_predicted, y_val)
        
        if(score>best_score):
            best_model = model
            best_score = score
            
        f1_scores.append(score)
        roc_auc_scores.append(roc_auc_score(y_predicted, y_val))
        accuracy_scores.append(accuracy_score(y_predicted, y_val))
        train_scores.append(model.score(X_train, y_train))
        model_names.append(model.__class__.__name__)
        
    model_report = pd.DataFrame({'F1 Score' : f1_scores, 
                                 'ROC-AUC Score' : roc_auc_scores,
                                 'Validation Accuracy Score': accuracy_scores,
                                 'Training Accuracy Score': train_scores,
                                 'Model' : model_names})
    
    model_report.set_index('Model', inplace=True)
    model_report.sort_values(by=['F1 Score','Validation Accuracy Score'], ascending=False, inplace=True)
    cache = [X_train, X_val, y_train, y_val]
    
    return cache, best_model, model_report
